# 🤖 NLP Chatbot using Hugging Face Transformers

## 📌 Task Overview
This task implements a simple conversational chatbot using Hugging Face Transformers and a pre-trained language model. The chatbot interacts with users, understands input text, and generates meaningful responses.

---

## 🚀 Features
- Conversational chatbot using pre-trained model
- Uses DialoGPT for response generation
- Handles real-time user input
- Includes basic rule-based responses (greetings, simple queries)
- Maintains conversation context

---

## 🛠️ Technologies Used
- Python
- Hugging Face Transformers
- PyTorch

---

## 🧠 How It Works
User enters input text
Text is tokenized using Hugging Face tokenizer
Input is passed to DialoGPT model
Model generates response based on conversation history
Response is decoded and displayed
