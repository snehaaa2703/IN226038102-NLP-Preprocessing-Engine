# BERT Text Classification (IMDB Dataset)

## Objective
Build a sentiment analysis model using BERT.

## Steps
- Data preprocessing
- Tokenization
- Model training using BERT
- Evaluation using accuracy and F1-score
- Experiments with freezing layers

## Results
The model achieved good accuracy on the IMDB dataset.
Full fine-tuning performed better than freezing layers.

## Tech Stack
- Python
- Hugging Face Transformers
- Scikit-learn
